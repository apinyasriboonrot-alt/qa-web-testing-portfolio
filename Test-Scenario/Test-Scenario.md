# SauceDemo Test Scenarios

## Module 1: Login

### SC-LOGIN-01
Verify user can login with valid credentials.

### SC-LOGIN-02
Verify system shows error when password is incorrect.

### SC-LOGIN-03
Verify system shows error when username is incorrect.

### SC-LOGIN-04
Verify system validation when username field is empty.

### SC-LOGIN-05
Verify system validation when password field is empty.

### SC-LOGIN-06
Verify user cannot login with locked account.


---

## Module 2: Product Listing

### SC-PROD-01
Verify product list is displayed after successful login.

### SC-PROD-02
Verify product details (name, price, image) are displayed correctly.

### SC-PROD-03
Verify sorting function (A-Z, Z-A, Price low-high, high-low).

### SC-PROD-04
Verify add to cart button is displayed for each product.


---

## Module 3: Cart Management

### SC-CART-01
Verify user can add product to cart.

### SC-CART-02
Verify cart badge updates correctly after adding product.

### SC-CART-03
Verify user can remove product from cart.

### SC-CART-04
Verify cart displays correct product information.


---

## Module 4: Checkout Process

### SC-CHECKOUT-01
Verify user can proceed to checkout.

### SC-CHECKOUT-02
Verify validation when required checkout fields are empty.

### SC-CHECKOUT-03
Verify checkout summary shows correct product and total price.

### SC-CHECKOUT-04
Verify order completion message after successful checkout.


---

## Module 5: Logout

### SC-LOGOUT-01
Verify user can logout successfully.

### SC-LOGOUT-02
Verify user cannot access dashboard after logout.
