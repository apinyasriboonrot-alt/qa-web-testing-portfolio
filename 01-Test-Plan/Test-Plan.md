# SauceDemo Test Plan

- Project Name: SauceDemo Website Testing
- Version: 1.0

To validate core functionalities of the SauceDemo web application and identify defects before production deployment.

### In Scope
- Login / Logout

- Add to Cart
- Remove from Cart
- Checkout

### Out of Scope
- Payment Integration
- Security Penetration Testing

## Test Type
- Functional Testing
- UI Testing
- Regression Testing
- Basic Automation (Login)

## Environment
- OS: Windows 10
- Browser: Chrome
- Device: Desktop

## Exit Criteria
- All test cases executed
- No critical defects remaining
